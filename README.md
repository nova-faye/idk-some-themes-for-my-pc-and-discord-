## 🚀 How to Install

All Discord themes require **[Vencord](https://vencord.dev)** or **[Equicord](https://equicord.org/)** with the **Visual Refresh** flag enabled.

1. Open Discord → **Settings** → **Vencord or Equicord** → **Themes**
2. Click **"edit quickcss"**
3. Paste the `@import` URL for the theme you want
4. Done — the theme applies instantly ✅

> **Tip:** You can also open the [Theme Browser](./themes_page.html) page in this repo for one-click copy buttons.

## 🎨 Discord Themes

All themes support both **Dark** and **Light** mode.

---

### 🟣 Catppuccin Mocha

> Classic purple & pink palette — a customized build of the official Catppuccin theme.

| Property | Value |
|---|---|
| Base | `#1e1e2e` deep navy |
| Accent | `#89b4fa` blue · `#cba6f7` lavender |
| Mood | Soft, pastel, cozy |
| Mode | Dark + Light |

**Install:**
```css
@import url("https://catppuccin.github.io/discord/dist/catppuccin-mocha.theme.css");
```
